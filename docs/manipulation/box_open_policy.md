# Box Open Policy

박스 개방 동작 sequence를 정리합니다.

---

## Motion Sequence

```text
접근 위치 조정
→ 손잡이 인식
→ 엔드이펙터 정렬
→ 전진 및 파지
→ 상승 동작
→ 지정 위치 이동
→ 파지 해제 및 복귀
```

---

## Checklist

```text
[ ] 상자 전체와 손잡이가 카메라 화면에 포함됨
[ ] 손잡이가 명확히 관찰됨
[ ] 엔드이펙터 좌우 위치 조정 완료
[ ] 파지 전 모터 부하 확인
[ ] 박스 개방 후 초기 자세 복귀
```
