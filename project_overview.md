## Project Overview

**ROScue**는 ROS2 기반의 지능형 협업 로봇 시스템입니다.  
로봇이 실내 공간을 탐색하며 폭발물 의심 상자를 탐지하고,  
모방학습 기반 매니퓰레이터 제어를 통해 상자를 개방한 뒤 내부 객체를 검사합니다.

폭발물 의심 객체가 감지되면 단일 로봇이 단독으로 처리하지 않고,  
파트너 로봇을 호출하여 **2대의 로봇이 협력하는 원격 해체 시나리오**를 수행합니다.

> 본 프로젝트는 실제 폭발물이 아닌 **교육용 모의 폭발물 객체**와 **모의 폭발물 상자**를 사용합니다.

<br>

| 구분 | 내용 |
|---|---|
| 프로젝트명 | ROScue |
| 주제 | ROS2 기반 지능형 협업 폭발물 해체 로봇 시스템 |
| 목적 | 위험 환경에서 사람의 직접 접근을 줄이는 로봇 협업 구조 구현 |
| 핵심 기능 | SLAM, 자율 탐색, YOLO 탐지, 박스 접근, 모방학습 기반 뚜껑 개방, 2로봇 협동 처리 |
| 주요 기술 | ROS2, Nav2, SLAM Toolbox, YOLO, Flask, Imitation Learning, RAG, LLM |
| 대상 객체 | 모의 폭발물 상자, 모의 폭발물 객체 |

---

### Key Features

- SLAM 기반 실내 지도 생성
- Nav2 기반 자율 탐색 및 목표 지점 이동
- YOLO 기반 폭발물 상자 및 내부 객체 탐지
- 모방학습 기반 박스 뚜껑 개방
- 중앙 서버 기반 로봇 상태 관리
- Web UI 기반 미션 제어 및 수동 조작
- RAG + LLM 기반 해체 절차 안내
- 2대 로봇 협동 처리 시나리오
